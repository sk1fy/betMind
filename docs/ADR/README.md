## Что такое ADR?

ADR (Architecture Decision Record) — документ, описывающий важное архитектурное решение:
- Контекст проблемы
- Рассматриваемые варианты
- Принятое решение
- Последствия

## Как добавить новый ADR

1. Создать файл `docs/ADR/XXX-title.md`
2. Использовать шаблон:
   - Статус (Proposed/Accepted/Deprecated)
   - Дата
   - Контекст
   - Рассматриваемые варианты
   - Решение
   - Последствия
3. Обновить этот индексный файл
