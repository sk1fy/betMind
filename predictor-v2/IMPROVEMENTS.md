## Приоритет 2: Улучшение моделей (2-3 недели)

### 2.1 Добавить новые модели в ансамбль

| Модель | Описание | Вес |
|--------|----------|-----|
| Head-to-Head | История личных встреч | 10% |
| Map Pool Diversity | Глубина пула карт | 5% |
| Recent Form Weighted | Взвешенная форма (последние 5 игр важнее) | 5% |

### 2.2 Улучшить существующие модели

**ExpertModel:**
- Добавить анализ формы на конкретной карте (не общей)
- Учесть время последней игры команды
- Добавить "усталость" при играх через день

**MomentumModel:**
- Учесть "выигрышные серии" (3+ победы подряд)
- Добавить анализ последних 5 матчей с весами

### 2.3 Новая модель: ML-модель

Добавить простую логистическую регрессию:
```python
from sklearn.linear_model import LogisticRegression

features = [
    rank_diff, 
    map_wr_diff, 
    form_diff, 
    odds_diff,
    pistol_wr_diff
```

### 3.2 Over/Under предсказания

Текущая проблема: постоянные "Меньше" (13:6, 13:7)

**Решение:**
- Добавить отдельную модель для тотала
- Учесть стиль игры команд (агрессивный/оборонительный)
- Добавить фактор "средний счёт на карте"
